# ZenTime

Time tracker and reporting tool. Simple project to explore features in dotnet core and React.

## Frontend

- React
- Typescript
- Redux
- Middelware error handling

## Backend

- AspNetCore API
- Logging
- HealthChecks
- Mediator CQRS
- Middleware error handling
- C# 8 nullable enabled
